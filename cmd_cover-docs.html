For instance, given S1 if cond { S2 } S3 counters will be added before S1 and before S3. The block containing S2 will be visited in a separate call. hasFuncLiteral

hasFuncLiteral reports the existence and position of the first func literal in the node, if any. If a func literal appears, it usually marks the termination of a basic block because the function body is itself a block. Therefore we draw a line at the start of the body of the first function literal we find. TODO: what if there's more than one? Probably doesn't matter much.

func hasFuncLiteral(n ast.Node) (bool, token.Pos)
