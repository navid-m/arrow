PackageName method

PackageName returns a name for the package being shown. It does this by choosing the penultimate element of the path name, so foo.bar/baz/foo.go chooses 'baz'. This is cheap and easy, avoids parsing the Go file, and gets a better answer for package main. It returns the empty string if there is a problem.
