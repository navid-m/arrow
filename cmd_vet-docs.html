errorCheck

errorCheck matches errors in outStr against comments in source files. For each line of the source files which should generate an error, there should be a comment of the form // ERROR "regexp". If outStr has an error for a line which has no such comment, this function will report an error. Likewise if outStr does not have an error for a line which has a comment, or if the error message does not match the . The syntax is Perl but it's best to stick to egrep. Sources files are supplied as fullshort slice. It consists of pairs: full path to source file and its base name.
